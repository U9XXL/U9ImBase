# U9ImBase

智能工厂移动应用-Base库，与[iUAP mobile](http://mobile.yyuap.com/)`summer框架`配合使用

## 安装

手动安装或者使用`bower`安装

```bash
$ bower install https://github.com/U9XXL/U9ImBase.git
```
